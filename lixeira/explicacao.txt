--------------------------------------------------
SISTEMA DE GERENCIAMENTO DE PADARIA - EXPLICAÇÃO
--------------------------------------------------

1. ESTRUTURAS DE DADOS
--------------------------------
1.1 data (armazena datas)
   - dia: dia do mês (1-31)
   - mes: mês do ano (1-12)
   - ano: ano com 4 dígitos

1.2 produto (cadastro de produtos)
   - codigoProd: código único (não se repete)
   - nomeProd: nome do produto (100 caracteres)
   - preco: preço unitário
   - estoque: quantidade disponível
   - nomeFornecedor: nome do fornecedor
   - foneFornecedor: telefone do fornecedor

1.3 funcionario (registro de funcionários)
   - matricula: número único de identificação
   - nomeFunc: nome completo
   - cargo: função na padaria
   - salario: valor do salário
   - login: nome de usuário
   - senha: senha de acesso

1.4 venda (registro de vendas)
   - codigoVenda: identificador único
   - prodVendido: código do produto vendido
   - vendedor: matrícula do funcionário
   - diaDaVenda: data da venda
   - quantidadeVendida: qtd. vendida

2. FUNCIONAMENTO DO SISTEMA
--------------------------------
2.1 Menu Principal
   - 1. Produtos (CRUD completo)
   - 2. Funcionários (CRUD completo)
   - 3. Vendas (em desenvolvimento)
   - 4. Sair

2.2 Operações CRUD
   - Create: Cadastrar novos itens
   - Read: Listar itens cadastrados
   - Update: Editar informações
   - Delete: Remover registros

3. FUNÇÕES PRINCIPAIS
--------------------------------
3.1 Para Produtos:
   - cadastrarProduto()
   - listarProdutos()
   - editarProduto()
   - excluirProduto()
   - buscarProduto()

3.2 Para Funcionários:
   - cadastrarFuncionario()
   - listarFuncionarios()
   - editarFuncionario()
   - excluirFuncionario()
   - buscarFuncionario()
